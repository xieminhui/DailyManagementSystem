uiGridScroller
service in module ui.grid.customScrolling

Description

uiGridScroller is an alternative to the native scroll event that uses touch events to ensure that grid scrolling works without a lag on devices.

Usage

uiGridScroller(element, scrollHandler);

Parameters

ParamTypeDetails
elementobject

Element being scrolled

scrollHandlerfunction

Function that needs to be called when scrolling happens.
